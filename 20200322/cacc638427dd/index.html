Windows 10 openSSH服务器

发表于2020-03-22更新于2021-05-15

windows 10 默认安装启用了openssh客户端，cmd和powershell中可以直接使用ssh命令连接到各种linux主机，但是windows 10 1909版本也提供选装的ssh服务端。

一、启用openssh服务端

打开设置-应用-应用和可选功能：

进入“可选功能”-“添加功能”-“OpenSSH服务器”-“安装即可”：

安装完成后以管理员权限运行cmd或powershell，运行命令net start sshd ：

然后在服务中打开OpenSSH SSH Server 和 OpenSSH Authentication Agent 服务，需要自动启动的，将其设置为自动

二、Windows下openssh服务端的使用

默认ssh进入的shell是cmd，可以运行命令powershell来使用powershell，考虑到powershell使用的频率很低，所以大部分情况下还是使用sftp传输文件。

推荐的sftp客户端：FileZilla
